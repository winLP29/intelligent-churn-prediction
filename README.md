# Intelligent Customer Churn Prediction

A **Flask web app** to predict customer churn for a Telco company using machine learning (Random Forest).  

---

## Features

- Predict customer churn probability
- Scales numerical features and one-hot encodes categorical features
- Visualizations for insights: tenure distribution, monthly charges, correlations
- Responsive web interface using Bootstrap
- Small dataset included (7,044 rows)
